readonly Color [] Corale.Colore.Razer.Keyboard.Effects.Custom._colors
private

Color definitions for each key on the keyboard.

The array is 1-dimensional, but will be passed to code expecting a 2-dimensional array. Access to this array is done using indices according to: column + row * Constants.MaxColumns

readonly Color [] Corale.Colore.Razer.Keyboard.Effects.Custom._keys
private

Color definitions for the key translation mode.

Colors set in here will, if flagged with 0x01000000, automatically be translated to the proper keyboard location depending on the users keyboard configuration.

Color Corale.Colore.Razer.Keyboard.Effects.Custom.this[Key key]
getset

Gets or sets the color for a specific key in the custom grid. The SDK will handle translation of location data to access the correct key depending on user configuration.

Parameters
keyThe Key to access.
Returns
The Color for the specified key.

Please note that the position of a key accessed in this way is not guaranteed to be correct, as different layouts on different keyboards can place these keys in other locations.
